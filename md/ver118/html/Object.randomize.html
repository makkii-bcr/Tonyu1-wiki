randomizeメソッド(Objectクラス)

rndメソッドが生成する乱数の系列(乱数の発生順序)を設定します。

書式
randomize(seed)
seed(省略可能) : 乱数系列を表す整数( 0 ～ 16777215 )。 省略すると、システムタイマによって乱数系列が設定されます。

このseedの値によって、この後rndメソッドが返す値の順番が決まります。 seedを省略した場合、プログラムを実行するたびにまったく違う乱数を返すことができます

例1

 実行方法... 
 
extends SpriteChar;
randomize();
while (1) {
  if( rnd(10)==0) {
    x=rnd()*$screenWidth;
    y=rnd()*$screenHeight;
  }
  update();
}

rndメソッド(Objectクラス)

乱数(でたらめな数)を返します
書式
rnd(n)
n(省略可能) : 発生させる乱数の範囲。

戻り値: 0 から n-1の整数乱数。nを省略すると0から1の実数乱数を返します。

例1

 実行方法... 
 extends SpriteChar;
while (1) {
  if( rnd(10)==0) {
    x=rnd()*$screenWidth;
    y=rnd()*$screenHeight;
  }
  update();
}
▲ 10フレームに１回の確率で、画面上のいろいろな場所にオブジェクトが移動します。

実行のたびに違う順序で乱数を発生させるにはrandomizeメソッドを使います。

Arrayクラス

Array(配列)クラスのオブジェクトには、複数の値（要素）を格納することができます。

例1

 実行方法... 
 
//予めコンソールウィンドウを表示しておいてください
extends SpriteChar;
a=new Array();   // 配列の作成
a.add("orange");
a.add("apple");
print ("要素数="+a.size());
print ("1番目の要素は"+a.get(0));
print ("2番目の要素は"+a.get(1));
 
a.insert(1,"banana");
a.set(0,"grape");
print ("要素数="+a.size());
print ("1番目の要素は"+a.get(0));
print ("2番目の要素は"+a.get(1));
print ("3番目の要素は"+a.get(2));
 
a.delete(2);
print ("全要素のリスト");
for (b in a) {
 print(b);
}
a.save("result.txt"); 
//files\result.txt に
//配列の内容が保存されます。

$Optionsクラスの説明に戻る

setメソッド($Optionsクラス)

プロジェクトの各種設定を行います。

書式
set(name,value)
name : 設定項目の名前
value : 設定する値

設定項目の名前(name)として指定できるのは、今のところ次いずれかです。

  • deactivation_pause : 値(value)が1ならば、Tonyuがアクティブでないときに実行を自動的に一時停止する。0なら一時停止しない。
  • deactivation_disableKey : 値(value)が1ならば、Tonyuがアクティブでないときにgetkeyメソッドを無効にする。0なら無効にしない。
  • sound_stopOnPageLoad : 値(value)が1ならば、$projectManager.loadPageメソッドでページを切り替えたときにBGMを停止する。0ならページをまたがってそのまま演奏される。

例1

 実行方法... 
 
extends SpriteChar;
$Options.set("deactivation_pause",1); 
//↑Tonyu以外のアプリケーションをアクティブにすると
//  一時停止する
while (1) {
   x++;
   update();
}

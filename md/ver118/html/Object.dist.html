distメソッド(Objectクラス)

線分(0,0)-(x,y)の長さを求めます。
書式
dist(x,y)
x : 線分(0,0)-(x,y)のx
y : 線分(0,0)-(x,y)のy

戻り値: 線分(0,0)-(x,y)の長さ

例1

 実行方法... 
 
extends SpriteChar;
while (1) {
  if ( dist(x-$teki.x, y-$teki.y)<50 ) die();
  update();
}
▲ オブジェクト $teki との距離が50以下になるとdie(); が呼ばれます。

参照： angle,sin,cos,distの関係

命令リファレンス
$TextEditorオブジェクト

$InputBoxより小型の、テキスト入力ウィンドウを表示します。

例1

 実行方法... 
extends SpriteChar;


$TextEditor.setPosition(10,10,200,100);
$TextEditor.setFont(11,"ＭＳ ゴシック");
$TextEditor.setText("何か入力してください");
$TextEditor.edit();
while($TextEditor.isEditing()) {
   update();
}
while (1) {
  drawText(x,y,$TextEditor.getText(),$clWhite);
  x++;
  update();
}

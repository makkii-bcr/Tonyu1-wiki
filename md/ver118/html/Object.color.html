colorメソッド(Objectクラス)

fillRect、drawLine、drawTextなどで、描画色を指定します

書式
color(r,g,b)
r : 赤成分(0-255)
g : 緑成分(0-255)
b : 青成分(0-255)

戻り値: 色を表す整数

例1

 実行方法... 
 
drawText(10,10,"赤巻き紙",color(255,50,50));
drawText(10,40,"青巻き紙",color(50,50,255));
drawText(10,70,"黄巻き紙",color(255,255,50));

また、次の8種類の色があらかじめ定義されています。 

変数名     色      colorメソッドによる定義
$clBlack   ■黒    color(  8,  8, 16)
$clRed     ■赤    color(255,  0,  0)
$clGreen   ■緑    color(  0,255,  0)
$clYellow  ■黄    color(255,255,  0)
$clBlue    ■青    color(  0,  0,255)
$clPink    ■紫    color(255,  0,255)
$clAqua    ■水色  color(  0,255,255)
$clWhite   □白    color(255,255,255)

※ $clBlackの色が中途半端なのは..?

例2

 実行方法... 
 
drawText(10,10,"赤巻き紙",$clRed);
drawText(10,40,"青巻き紙",$clBlue);
drawText(10,70,"黄巻き紙",$clYellow);

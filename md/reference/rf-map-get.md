## $map.get

マップパターンを取得します

#### 書式
```
get(mx,my)
```

- **mx**
&emsp;マップパターン上のx座標
- **my**
&emsp;マップパターン上のy座標

#### 戻り値
マップ上のmx,myにおけるキャラクタパターン番号を返します


### 例1

```
 pa=$map.get(4,3);
```

## 真偽

[if](./rf-if)文や[while](./rf-while)文などに用いられる条件式は、その条件が成立したか否かを表す必要があります。 条件が成立したときを**真**、成立していない場合を**偽**といいます。

Tonyuでは、すべての値が「真」または「偽」を表すようになっています。

- 真を表す値： 0以外の数値、オブジェクト(文字列も含む)
- 偽を表す値： 数値の0、null

例

- 5>3 : 真
- 1+1==3 : 偽
- x<10 : xが10未満の数値なら真、それ以外の場合は偽
- 1 : 真
- 0 : 偽
- "tonyu": 真

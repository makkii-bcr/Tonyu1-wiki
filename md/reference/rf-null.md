## null

**null**とは、「何も値が入っていない」という意味です。変数に値が代入される前には、このnullという値が入っています。

ある変数がnullであるか(まだ値が入っていないか)どうかは、次のように判定できます

```
if (x==null) ...
```

次のように、変数にnullを代入することも可能です。

```
x=null;
```

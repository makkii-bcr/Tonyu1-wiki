## $map.set

マップパターンを設定します

#### 書式
```
set(mx,my,p)
```

- **mx**  
&emsp;マップパターン上のx座標
- **my**  
&emsp;マップパターン上のy座標
- **p**  
&emsp;設定するキャラクタパターン番号

マップの(mx,my)のパターンをpに設定します

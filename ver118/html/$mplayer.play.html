playメソッド($mplayerクラス)

BGM,効果音を演奏します
書式
play(s,autoReplay,vol)
s : 演奏するサウンドオブジェクト
autoReplay(省略可能) : Midiを繰り返し演奏する場合は1
vol(省略可能) : 効果音の音量(0-128)

 実行方法... 
 
/*(予め、BGM・効果音ウィンドウで
test.wavというサウンドファイルを読み込んでおく必要があります)*/
extends SpriteChar;
while (1) {
  if (getkey(32)==1) {$mplayer.play($se_test);}
  update();
}

PanelCharクラス

任意の図形（線、四角形、文字などの組み合わせ)が描けるオブジェクトです

PanelCharクラスのオブジェクトを作るには、図1のように「パネル」を選びます

図1 :

変数panelを通じて、Panelクラスに用意されている描画命令を使うことができます。

例1

 実行方法... 
 
extends PanelChar;
resize(200,200);
while (1) {
  x=x+1;
  panel.drawLine(0,0,200,t,$clWhite);
  t=t+1;
  update();
}
